# Rock, Paper, Scissors Game (Python)

A classic interactive Rock-Paper-Scissors game played against the computer. This project demonstrates the use of random number generation, user input handling, and game logic in Python.

## 🚀 Features
* **Computer AI:** Uses Python's `random` module for unpredictable moves.
* **Score Tracking:** Keeps a real-time tally of player vs. computer wins.
* **Input Validation:** Handles case-insensitive inputs (e.g., "Rock", "ROCK", "rock") and invalid entries.
* **Safe Exit:** Allows the player to quit anytime by pressing 'q' and see the final results.

## 🛠️ Technologies Used
* **Language:** Python 3.x
* **Standard Library:** `random`
* **Programming Concepts:** Functions, while loops, conditional statements, and string formatting.

## 💻 How to Run
1. Ensure Python 3 is installed.
2. Clone this repository or download `rock_paper_scissors.py`.
